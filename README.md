СКРИПТ МОЖЕТ НЕ РАБОТАТЬ В НЕКОТОРЫХ МЕСТАХ
Скрипт является просто первой попыткой в программировании

Он поможит настроить Arch linux/EndeavourOS/OpenSUSE Tumbleweed(Старый, не тестированый код)


# Зависимости
fastfetch
lsb-release

# Использование
